# Birthday Website

A simple static birthday website for Charvi.

## Features
- Passcode-protected landing screen
- Reveal animation with confetti and a birthday melody
- Timeline section with placeholders for photos
- Final message section for birthday wishes

## How to use
1. Open `index.html` in a browser.
2. Enter passcode `30-05-2006`.
3. Click "Open your celebration" and then "Proceed further".
4. Replace the placeholder timeline cards with real photos and captions.
